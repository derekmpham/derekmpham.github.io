Another website about the Lakers is Lakers Nation, a blog site about the latest news on the Lakers.

What makes this site good is its extensive use of different images and pages. Lakers Nation uses a different image to complement each and every blog. Moreover, the site directs the user to a new page if the user wants to read further on a specific blog--the use of an entirely new page for every blog creates a more comfortable user experience.
